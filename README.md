##  Text to Image Generator (React)
A simple AI-powered image generator that converts your text prompts into beautiful images using the Pollinations API.


###  Features
- Generate AI-based images from text prompts  
- Change image background color dynamically  
- Adjust image width and height  
- Download generated images  
- Responsive and user-friendly UI
